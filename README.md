# Vue 3 + Vite + Tailwind CSS


```
npm create vite@latest my-project -- --template vue
cd my-project
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
npm run dev
```

source: https://tailwindcss.com/docs/guides/vite#vue
